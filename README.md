# MintVelvetAutoDemo
## Mint Velvet Automation Demo

**Framework highlights**

- Webdriver Selenium
- TestNG as the unit testing framework
- Extended Report
- Parallel Run
- Page Object Model
- Multi-browser run

- API testing using RestAssure libraries

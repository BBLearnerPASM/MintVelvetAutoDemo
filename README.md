# MintVelvetAutoDemo
## Mint Velvet Automation Demo ##

**Framework highlights**
1         Webdriver Selenium
2         TestNG as the unit testing framework
3         Extended Report
4         Parallel Run
5         Page Object Model
6         Multi-browser run

- API testing using RestAssure libraries
